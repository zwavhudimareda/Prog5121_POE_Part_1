ChatBeans – Login & Registration System
Overview:

ChatBeans is a simple Java console application that allows users to register and log in. The system validates user input to ensure that usernames, passwords, and cell phone numbers meet specific requirements.

Features:
User registration with input validation
Username validation (must contain "_" and be ≤ 5 characters)
Password complexity validation (minimum 8 characters, capital letter, number, special character)
South African cell phone number validation (+27 format)
Login authentication
Immediate feedback for incorrect inputs
Unit testing using JUnit
